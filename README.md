# Hackathon

### Prepearing and running client:

#### Go to client folder,
#### Install modules:

```
npm i 
```

#### Build app:

```
npm run build 
```

#### Run client:

```
npm run start
```

### Prepearing and running server:

#### Go to server folder

#### Rename 

```
.env-sample file to .env
```
#### Install modules:

``` 
npm i
```

#### Build server:
```
npm run build
```
#### Run server: 
```
npm run start
```
