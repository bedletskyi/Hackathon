# Hackathon

#### Go to server folder
#### Rename 

```
.env-sample file to .env
```
### Prepearing and running client:

Now you can run the project using CLI or docker. Go to **scripts** folder if you want to run in docker 
#### Go to client folder,
#### Install modules:

```
npm i 
```

#### Build app:

```
npm run build 
```

#### Run client:

```
npm run start
```
