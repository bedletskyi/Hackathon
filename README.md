# Hackathon

Prepearing and running client:

Go to client folder,
Install modules:

```
npm i 
```

Build app:

```
npm run build 
```

Run client:

```
npm run start
```

Go to client folder

Rename .env-sample file to .env

Prepearing and running server:

``` 
npm install -g nodemon
npm i
```

Run server:
```
npm run hot-server 
```
